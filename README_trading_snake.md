# 量化交易贪吃蛇系统 - Quantitative Trading Snake System

## 🎮 游戏概述

这是一个结合了贪吃蛇游戏机制的量化交易系统，通过游戏化的方式模拟量化交易策略的执行过程。

## 🚀 安装依赖

```bash
pip install pygame numpy
```

## 🎯 游戏机制

### 🐍 交易蛇
- **移动控制**: 使用方向键控制交易蛇移动
- **成长机制**: 吃到盈利机会时蛇身增长，亏损时蛇身缩短
- **颜色变化**: 蛇身颜色根据总盈亏变化（盈利=绿色，亏损=红色）

### 💰 交易机会
- **绿色 ($)**: 盈利机会 (+$10 到 +$50)
- **红色 (L)**: 亏损机会 (-$30 到 -$10)
- **橙色 (B)**: 突破机会 (+$20 到 +$80)
- **紫色 (R)**: 反转机会 (+$30 到 +$100)
- **青色 (S)**: 剥头皮机会 (+$5 到 +$25)
- **深紫 (W)**: 摆动交易机会 (+$40 到 +$120)

### 📊 市场环境
- **牛市 (BULL)**: 更多盈利机会
- **熊市 (BEAR)**: 更多亏损和反转机会
- **横盘 (SIDEWAYS)**: 均衡机会分布
- **波动市 (VOLATILE)**: 更多突破和剥头皮机会

## 🎮 操作说明

### 基本操作
- **↑↓←→**: 控制交易蛇移动
- **P**: 暂停/继续游戏
- **ESC**: 返回主菜单/退出

### 游戏流程
1. 主菜单按空格键开始交易
2. 控制蛇吃各种交易机会
3. 监控市场指标和交易统计
4. 管理风险和头寸大小
5. 游戏结束查看最终绩效

## 📈 核心特性

### 🏦 量化分析
- **实时价格生成**: 基于随机游走模型
- **技术指标**: MA20/50, RSI, 波动率计算
- **市场条件识别**: 自动判断牛熊震荡市
- **趋势跟踪**: 动态计算市场趋势

### 💼 交易管理
- **风险控制**: 基于波动率的动态风险评分
- **头寸管理**: 根据风险调整持仓大小
- **盈亏跟踪**: 实时P&L计算和显示
- **绩效指标**: 胜率、最大回撤、夏普比率等

### 🎯 游戏化元素
- **难度递增**: 根据表现动态调整难度
- **成长机制**: 蛇身长度代表交易成功
- **视觉反馈**: 颜色和动画反映市场状态
- **成就系统**: 最高分记录和新纪录提示

## 📊 统计指标

### 交易绩效
- **总资本**: 实时账户余额
- **总盈亏**: 累计损益
- **胜率**: 盈利交易占比
- **交易次数**: 总交易笔数
- **最大回撤**: 资金最大跌幅
- **风险评分**: 综合风险评估

### 市场指标
- **当前价格**: 模拟市场价格
- **波动率**: 价格波动程度
- **RSI**: 相对强弱指数
- **市场条件**: 当前市况

## 🔧 策略逻辑

### 机会生成
- 基于市场条件加权生成交易机会
- 高风险高收益和低风险低收益的平衡
- 动态调整机会出现频率

### 风险管理
- 高风险交易减少头寸规模
- 低风险交易增加头寸规模
- 连续亏损自动收缩头寸

### 自适应系统
- 根据交易表现调整游戏难度
- 市场环境变化影响机会分布
- 实时更新技术指标

## 🎯 教育价值

1. **风险管理**: 通过游戏理解风险控制重要性
2. **策略适应**: 学会根据市场环境调整策略
3. **交易心理**: 体验连续盈利/亏损的心理影响
4. **技术分析**: 直观理解技术指标的应用
5. **资金管理**: 学习仓位控制和资金分配
